**************** Pré-requis Materiel ****************
- Ordinateur avec au moins 8Go de RAM
- Serveur
- Telephone android avec au moins 2Go de RAM et 5"

**************** Pré-requis logiciel ****************
- Android studio avec Gradle version 7.3 au moins
- XAMPP Server ou installer un serveur web et de BD
- Un navigateur

************* Procedure d'installation **************
1) Installer Android studio, Telecharger un emulateur et un SDK 12 sur Android Studio
2) Decompresser le fichier code-manu.rar, et l'importer comme projet dans Android Studio
3) Decompresser le fichier api-php-manu.rar et copier son contenu dans le serveur web (C:\xampp\htdocs pour Xampp)
4) Ouvrir son SGBD qui prend en charge SQL et y importer le fichier bdmanu.sql
5) Lancer le deboguage de l'application
